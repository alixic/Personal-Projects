Meniu Despre Functionare Componente Utilizare Imagini Formular

        Componente interne din cadrul Flipper Zero

 

        1. Placa de baza (cablajul principal) compusa dintr-un microcontroller STM32, display-ul monocrom, mufa de alimentare USB-C, modul radio sub 1GHz si butoane.

            Microcontroller-ul STM32 este bazat pe arhitectura ARM, ce incorporeaza doua nuclee, 256KB de RAM si 1024KB de memorie(stocare) in Flash. (STM32WB55)

        2. PCB IButton : Aceast cablaj are pini pentru iButton, buzzer si pentru un transmitator-receptor infrarosu.

        3. PCB NFC+RFID : Este o alta sectiune de cablaj ce integreaza componentele necesare tehnologiilor RFID (pe 125kHz) si NFC (pe 13.56 MHz).

        4. PCB cu antena : Utilizat pentru tehnologiile NFC si RFID, localizata sub intregul cablaj / sistem.

        5. Carcasa pentru stocarea si protejarea componentelor de praf, umiditate,etc.

        6. Baterie de 2100 mAh.

        7. GPIO headers.

        8. Slot card MicroSD.

        9. Motor pentru vibratii.
