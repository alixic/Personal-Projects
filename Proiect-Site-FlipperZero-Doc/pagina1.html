                  Despre Flipper Zero

 

        Flipper Zero este un dispozitiv portabil ce are integrate mai multe functii utile ce ii permit interactiunea cu sisteme de control al accesului. Acesta este capabil de a citi, copia si imita (simula) tag-urile RFID si NFC, telecomenzi radio, iButton si chei digitale de acces prin utilizarea pinilor de tip GPIO.

    Este unul dintre cele mai utilizate dispozitive de catre expertii in securitate cibernetica ce fac pentesting, deoarece permite procesul de hacking asupra protocoalelor radio, sistemelor de control al accesului, la nivel de hardware.

    Fiind open-source, acest fapt a incurajat promovarea si popularizarea dispozitivului Flipper Zero. De altfel, este un echipament customizable, care poate fi extins si imbunatatit pentru a mari gama de functii ce le poate furniza.

     Ideea de Flipper Zero prezinta combinarea tuturor instrumentelor hardware de care poti avea nevoie pentru explorare si dezvoltare in miscare (portabilitate). Flipper a fost inspirat de proiectul pwnagotchi, dar spre deosebire de alte placi DIY, are o carcasa robusta, butoane la indemana si o carcasa ce ofera protectie cablajului si a pinilor.
